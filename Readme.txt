# 🌐 Software Company Website

A modern, responsive landing page for a software company that offers GTM-based solutions, warehouse management, vendor management, and product management services.

## 🚀 Project Overview

This is a **static front-end project** built using **HTML** and **CSS**, with a placeholder for JavaScript interactions. The design includes:

- Hero section with company introduction
- Navigation bar
- Services section (WMS, VMS, PMS)
- Company locations with hover effects

## 💻 Technologies Used

- **HTML5** for page structure
- **CSS3** for styling and layout
- **JavaScript** (empty `script.js` file as a placeholder)
- **Google Fonts** for typography

## 🧩 Features

- Fully responsive layout
- Animated hover effects on navigation and company location cards
- Clean, modern design using gradient overlays and card-based sections
- Hero call-to-action for user engagement

## 🌟 Sections Breakdown

- **Header + Navigation Bar**: Contains logo and menu links
- **Text Box (Hero Section)**: Introduces the company and its GTM service offerings
- **Course Section**: Lists the services WMS, VMS, PMS
- **Company Section**: Highlights different office locations with hover overlays
